# Projeto SQL - Clientes e Pedidos

Projeto desenvolvido para praticar conceitos de banco de dados relacionais utilizando SQL Server.

## Funcionalidades

* Criação de tabelas
* Chaves primárias e estrangeiras
* Inserção de dados
* Relacionamento entre clientes e pedidos
* Consultas utilizando JOIN
* Relatórios com COUNT, AVG e SUM

## Conceitos aplicados

* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* INSERT INTO
* INNER JOIN
* GROUP BY
* ORDER BY
* COUNT
* AVG
* SUM

## Objetivo

Praticar modelagem de banco de dados e consultas SQL para análise de pedidos e faturamento.
